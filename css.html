CSS (Cascading StyleSheet)

“CSS is one of the core languages of the open web and is standardized across browsers according to the W3C specification” -developer.mozilla.org. It was developed by Håkon Wium Lie Bert Bos. CSS is a language used for designing a document. It is used widely by web developers to create a visual engaging webpages, user interfaces for web applications, and user interfaces for many mobile

CSS STATEMENTS

CSS Rules Set - A rule set is a statements that tells browsers how to render particular elements on an HTML page. It consists of a selector followed by a declaration block.

CSS Rule Precedence - an HTML element may be the subject of the selectors of multiple stylesheets. when such values target different properties, their effects cascade and when the styles involve the same property, they conflict.

CALCULATING A SELECTOR’S SPECIFICITY

  • 1. Important rules override normal rules.
  • 2. Count the number of ID selectors in the selector. (=a)
  • 3. Count the number of class selectors, attribute selectors and pseudo-classes in the selector. (=b)
  • 4. Count the number of type selectors and pseudo-elements in the selector. (=c)
  • 5. Ignore the universal selector.

CSS Selectors and Combinators

Selector – structure used as a condition in a CSS rule to determine which elements in the document tree are matched by the selector.

Selector syntax: Chain of one or more sequences of simple selectors.

  • - Not separated by combinators.

Selector group - comma-separated list of selector.

CSS Values

CSS values are set against CSS Properties and reside within CSS declaration block, which is a part of the CSS rule / statement.
CSS 2.1 allows following types of values :

  • Integers and real numbers
  • Lengths
  • Percentages
  • URLs and URIs
  • Counters
  • Colors
  • Strings

Integers and real numbers

Integers and real numbers can be used as CSS values. When used, they can be preceded by + or - sign. Decimal numbers are also allowed. Used in following CSS properties:

  • z-index
  • line-height
  • counter-increment
  • used as CSS color values

Lengths

Lengths can be used to specify horizontal or vertical measurements. When used, length is followed immediately by a unit. There should not be any space between the number and the unit. For certain properties, negative lengths are allowed , but for certain properties using a negative value results from an error. Used in following CSS properties:

  • width
  • margin
  • padding
  • font-size
  • border-width
  • text-shadow

Percentages

Percentages can be used by specifying a number followed by a "%". Used in following CSS properties:

  • width
  • margin-top
  • font-size

URLs and URIs

URLs and URIs are used as CSS values to point to a resource ( e.g. an image ). The format of a URI value is url("URL" ). The quotation around the URL is optional. You can also use Relative URIs. Used in following CSS properties:

  • background image
  • cursor
  • list-style

Counters

Counters are denoted by case-sensitive identifiers. The format of using counters are counter(counter-identifier). A common example of a counter identifier is per-num. Can be used with most if the element selectors and pseudo selectors.

Colors

Acolor is either a keyword or a numerical RGB specification or a hexadecimal color value. It is used to set the color of the text written within an HTML document. Used in color, background, border or box-shadow are common examples of CSS properties that use colors as a value. Most of the CSS properties accept color as values.

Strings

The format of writing strings may be either with double quotes or with single quotes. Used in most of the CSS properties accept strings as values.
